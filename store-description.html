🚀 Coze知识库导入助手

一键将网页内容导入到Coze知识库，让AI知识管理更智能、更高效！

✨ 核心特性

  • 🎯 一键导入 - 快速保存当前网页内容到Coze知识库
  • 📚 智能分段 - 自动优化长文本，确保AI理解更准确
  • 🔒 安全可靠 - 所有配置信息仅保存在本地，确保数据安全
  • ⚡ 快捷操作 - 支持快捷键（Ctrl+Shift+S）快速启动
  • 🎨 简洁界面 - 清晰直观的操作界面，零学习成本

🎯 适用场景

  • 📝 收集文章和博客内容到知识库
  • 📊 保存重要的技术文档和教程
  • 🔍 整理研究资料和参考文献
  • 💡 收集灵感和创意素材

📖 使用方法

  1. 配置Access Token
    • 访问Coze开发者设置页面
    • 创建并复制Access Token
    • 粘贴到插件配置中
  2. 设置知识库ID
    • 在Coze平台选择或创建知识库
    • 复制知识库ID
    • 填入插件配置
  3. 开始使用
    • 浏览任意网页时点击插件图标
    • 点击"导入当前页面"即可
    • 或使用快捷键 Ctrl+Shift+S（Mac: Command+Shift+S）
