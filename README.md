# CozeKnow - Coze知识库导入助手

这是一个Chrome浏览器扩展,可以方便地将当前网页内容导入到Coze知识库中。

## 功能特性

- 支持将当前浏览的网页内容一键导入到Coze知识库
- 可配置Coze access token和知识库ID
- 数据安全存储在本地
- 简单易用的操作界面

## 使用说明

1. 安装扩展后,点击工具栏中的扩展图标
2. 首次使用需要配置:
   - Coze Access Token
   - 知识库ID
3. 配置完成后,在任意网页点击扩展图标即可将当前页面导入到知识库

## 隐私说明

- 所有配置信息仅存储在本地浏览器中
- 不会将您的token等敏感信息上传到任何第三方服务器
